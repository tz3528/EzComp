## 编译器命令

- `ezcomp <file> --emit=ast` — 输出 AST
  ```
  ezcomp ../test/compare/2d_heat_equation/2d_heat_equation.comp --emit=ast
  ```

- `ezcomp <file> --emit=mlir` — 输出 MLIR (Comp Dialect)
  ```
  ezcomp ../test/compare/2d_heat_equation/2d_heat_equation.comp --emit=mlir
  ```

- `ezcomp <file> --emit=llvmir` — 输出 LLVM IR
  ```
  ezcomp ../test/compare/2d_heat_equation/2d_heat_equation.comp --emit=llvmir
  ```

- `ezcomp <file> --emit=compile` — 编译为可执行文件
  ```
  ezcomp ../test/compare/2d_heat_equation/2d_heat_equation.comp --emit=compile
  ```

- `ezcomp <file> --mcpu=<cpu>` — 指定目标 CPU 架构
  ```
  ezcomp ../test/compare/2d_heat_equation/2d_heat_equation.comp --emit=compile --mcpu=skylake
  ```

- `ezcomp <file> --mattr=<features>` — 指定 CPU 特性 (如 +avx2,+fma)
  ```
  ezcomp ../test/compare/2d_heat_equation/2d_heat_equation.comp --emit=compile --mattr=+avx2,+fma
  ```

- `ezcomp <file> --pass-pipeline=<pipeline>` — 自定义降级管线
  ```
  ezcomp ../test/compare/2d_heat_equation/2d_heat_equation.comp --emit=mlir --pass-pipeline="builtin.module(lowering{comp-base=true})"
  ```

## Pass Pipeline 实例

- Comp → Base 降级
  ```
  ezcomp ../test/compare/2d_heat_equation/2d_heat_equation.comp --emit=mlir --pass-pipeline="builtin.module(lowering{comp-base=true})"
  ```

- Comp → Affine → SCF 降级
  ```
  ezcomp ../test/compare/2d_heat_equation/2d_heat_equation.comp --emit=mlir --pass-pipeline="builtin.module(lowering{comp-base=true affine-scf=true})"
  ```

- 完整四阶段降级 (Comp → Affine → SCF → CF → LLVM)
  ```
  ezcomp ../test/compare/2d_heat_equation/2d_heat_equation.comp --emit=mlir --pass-pipeline="builtin.module(lowering{comp-base=true affine-scf=true scf-cf=true to-llvm=true})"
  ```

## 测试命令

- `cmake --build build --target check` — 运行集成测试

- 比较测试

  **命令格式：**

  - Linux/macOS:
    ```
    COMPARE_TEST_DIR=<test_name> cmake --build build --target compare
    ```

  - Windows (PowerShell):
    ```powershell
    $env:COMPARE_TEST_DIR="<test_name>"; cmake --build build --target compare
    ```

  **命令实例：**

  - Linux/macOS:
    ```
    COMPARE_TEST_DIR=2d_heat_equation cmake --build build --target compare
    ```

  - Windows (PowerShell):
    ```powershell
    $env:COMPARE_TEST_DIR="2d_heat_equation"; cmake --build build --target compare
    ```
